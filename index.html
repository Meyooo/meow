Введение

Чтобы создавать веб-сайты, вы должны знать о HTML ( HyperText Markup Language / Язык гипертекствовой разметки ) - фундаментальной технологии, которая используется для определения структуры веб-страницы.

HTML применяется для того, чтобы опредлеить как должен отображаться ваш контент в различном виде:

  • Абзац
  • Список
  • Заголовок
  • Ссылка
  • Изображение
  • Мультимедиа
  • Формы

Существует и множество других доступных элементов, также вы можете создать свой собственный новый элемент.

Темы для изучения

Для того, чтобы создать свою первую страницу с помощью HTML нужно понимать некоторые темы:

  1. Работа с текстом
  2. Атрибуты
  3. Ссылки
  4. Изображения
  5. Таблицы
  6. Формы
